
              REPSTA

Модули программы:

1) Модуль парса ключей
2) Модуль интерфейса графики
3) Модуль сбора информации


    1. Модуль парса ключей (Keys) [PROGRESS 10%]
  Модуль содержит терминальный интерфейс программы, который принимает
  ключи для обработки флагов.
  
  Программа содержит следующие ключи:
    - --help, -h - показывает список ключей и информацию о программе
    - --version, -v - показывает текущую версию программы
    - --logfile, -l - сохраняет отчёт по указаному пути
    - --date, -d - показывает дату создания репозитория
    - --creator, -c - показывает имя создателя репозитория
    - --fast, -f - показывает краткую информацию о репозитории
  Список использованных ключей: {h,v,l,d,c,f};

    2. Модуль интерфейса графики (TUI) [PROGRESS 0%] 
  Модуль реализован через библиотеку ncurses.
  Экран представляет собой лист файлов в репозитории в центре экрана,
  над листом файлов находится краткая сводка о репозитории, при нажатии на
  F1 откроется окно со справкой комманд. При нажатии на ENTER на файле,
  слева вылетит панель с подробной информацией о файле.
